Environment: Windows
	


Assumptions:
    Customizable grid size (1 to 26).
    Ability to set the number of mines (up to 35% of total squares).
    Interactive gameplay with feedback on the grid.
    Automatic uncovering of adjacent squares when a cell with zero adjacent mines is revealed.
    Win and lose conditions based on gameplay.

Java Class Overview
    Game: Manages the game flow, user input, and displays the grid.
    Grid: Represents the game board, generates mines, and handles cell revelations.
    Cell: Represents each cell in the grid, stores information about mines and adjacent mine counts.


How to Run
	Open Eclipse
	Open Project
	Right Click on App.java >Run As > Java Application
